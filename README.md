# Oriz Shift — Universal Converter

Shift converts and changes units, formats, and values — from length, weight, and temperature to currencies, timezones, file formats, and encodings. One unified surface for every "from X to Y" conversion.

**Stack:** Astro 6 · React 19 · Tailwind v4
**Live:** https://convert.oriz.in
