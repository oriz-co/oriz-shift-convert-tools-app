# Oriz Shift — Convert tools

> Universal converter — shifts and changes units, formats, and values across every "from X to Y" domain.

**Live at**: <https://convert.oriz.in> · **Status**: scaffold

## What this is

One unified surface for every conversion: length, weight, temperature, currencies, timezones, file formats, encodings, and beyond. If it converts from X to Y, it belongs here.
